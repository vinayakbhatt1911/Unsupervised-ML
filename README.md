# Unsupervised-ML

🧠 Unsupervised Machine Learning
This repository contains beginner-friendly implementations of Unsupervised Machine Learning algorithms using Python and Scikit-learn. Each algorithm is explained with a biotechnology-themed mini project and data visualization using Matplotlib.
📌 Algorithms Covered
K-Means Clustering
Hierarchical Clustering
DBSCAN (Density-Based Spatial Clustering)
📂 Mini Projects
1️⃣ K-Means Clustering
Project: Bacterial Colony Clustering
Objective: Cluster bacterial colonies based on colony diameter and growth rate without using labeled data.
2️⃣ Hierarchical Clustering
Project: Microbial Species Clustering
Objective: Group microbial species into clusters based on cell size and growth rate using hierarchical clustering and visualize the relationships using a dendrogram.
3️⃣ DBSCAN
Project: Outlier Detection in Bacterial Growth
Objective: Identify bacterial growth clusters and detect abnormal (noise) colonies using density-based clustering.
